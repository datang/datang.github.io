把windows10打造成软件开发的神器

发表于 2018-05-28 |

工欲善其事，必先利其器。对于软件工程师来说有个舒服的coding环境真的太有必要了。一台Windows主机加上一台Ubuntu主机，然后使用神器Synergy来实现一套键盘/鼠标控制，这是我最舒服的方式。

跟了我多年的x200终于退役了，家里添置了新的台式电脑。那么问题来了，只有一台主机，要同时满足Windows+Ubuntu的需求应该怎么办呢？中间纠结的过程就不细说了，最终的解决方案是安装最新版Windows10系统，使用自带的Hyper-V安装Ubuntu Server 18.04虚拟机，再综合运用各种工具软件（Docker、PuTTY）。最终效果非常不错，Hyper-V的表现超出预期。

都说Mac OS是最适合程序员使用的操作系统，那么，我们就通过接下来的一波操作，打造最适合程序员的Windows系统。

友好的Windows10

Hyper-V、PowerShell、Subsystem-Linux等等，Windows对程序员真的是越来越友好了。更何况日常办公Windows下各种应用软件体验也是非常的不错，没必要做Linux的卫道士，博采众长才是硬道理。

超出预期的Hyper-V

曾经使用过VirtualBox、VMware，都是非常不错的虚拟化软件，如果是在虚拟机安装Windows或者Linux的桌面版，这两个都是不错的选择。当然，VMware是收费的，免费的可以用VMware Player。

这里要使用虚拟机安装Ubuntu Server，Hyper-V无意是首选。资源占用率极低，性能强，速度快，最重要的是：

Hyper-V支持宿主机关机时虚拟机自动保存，宿主机开机时虚拟机自动开启。

也就是说你开机后就直接可以通过PuTTY登录到Ubuntu Server虚拟机，就像连接一个远程的永不宕机的Linux服务器一样，真是毫无存在感。
